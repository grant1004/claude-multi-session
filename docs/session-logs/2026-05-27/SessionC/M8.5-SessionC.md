   - Why: Audit operates on main. The session branch is created but the Reviewer stays on main — Workers will branch from the session branch when worktrees are created.

5. **§9 report updated to include session branch name**
   - Why: Reviewer needs the branch name to create worktrees (`git worktree add ../worker-<id> -b worker/<id> session/<slug>`).
