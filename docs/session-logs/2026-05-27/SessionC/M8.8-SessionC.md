2. **Session branch auto-detection with fallback**
   - Why: Self-check needs to know the diff target. `git branch --list 'session/*'` finds it automatically. Multiple session branches handled via `merge-base --is-ancestor`. Falls back to `main` with ⚠️ if no session branch — ensures backwards compatibility.

   - Why: Workers branch from the session branch, not main. Diffing against main would show changes from other milestones already merged to the session branch, producing false file-scope violations.
